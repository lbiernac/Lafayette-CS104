# Lafayette-CS104

Welcome! This GitHub Repository records the code examples used in the course CS 104-03 (Computer Gaming) at Lafayette College in Fall 2024. This repository contains code examples written with Processing 4.3 and with Java. Below are some helpful links for getting started with Processing. 

**Instructions for Downloading Processing:**
https://processing.org/tutorials/gettingstarted

**Getting your Bearings on the Processing IDE:** 
https://processing.org/environment/
